Week 2 Notes

Variables

The keyword const is used when the variable will not be reassigned to another value, whereas let is used if the variable might be reassigned later in the program.

Arrays

An array is an ordered list of values. It can help us find something. It is created by using "[]" and the elements inside can be separated by commas.
When we define an array, the list count starts width 0 and moves up. Here's an example of an array:
const fruits = ['pineaple', 'strawberry', 'lime']
