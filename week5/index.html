Testing and Debugging

Errors
These are caused when something goes wrong in a program. Can be System errors, programmer errors and user errors.

Exceptions
Is an error that produces a return value that can be used by the program to deal with the error

Stack Traces
Is a sequence of funcions or method calls that lead to the point where the error occurred.

Warnings
It can occur if there's an error in the code that isn't enough to cause the program to crash
